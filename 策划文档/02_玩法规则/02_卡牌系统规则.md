# 卡牌系统规则

## 第一版卡牌范围

第一版只开放 3 个核心流派：

```text
温压弹
干冰弹
电磁穿刺
```

所有核心技能卡牌遵循：

```text
打出该流派卡牌时，先执行该流派基准效果，再叠加该卡牌的词条效果。
```

---

## 手牌与补牌

推荐第一版参数：

| 参数 | 数值 |
|---|---:|
| 自动补牌目标 | 4 |
| 开局抽牌 | 抽满 3 张 |
| 打出卡牌后 | 卡牌进入弃牌堆 |
| 补牌方式 | 手牌低于自动补牌目标时自动补牌 |
| 补牌间隔 | 1.5 秒 / 张 |
| 牌库空时 | 弃牌堆洗回牌库 |

说明：

- 自动补牌目标不是硬性手牌上限。
- 主动抽牌、未来的回手或复制获得可以让手牌超过自动补牌目标 4 张。
- 三选一获得新卡时，如果当前手牌低于自动补牌目标 4 张，新卡直接进入手牌；如果当前手牌已经达到或超过 4 张，新卡进入牌堆。

弃牌按钮：

```text
点击后弃掉当前全部手牌。
立即补牌至自动补牌目标。
冷却 20 秒。
弃牌会中断当前连锁。
```

## 出牌流程

```gdscript
func try_play_card(card_id: String) -> bool:
    var card = ConfigDB.get_card(card_id)

    if energy.current < card.cost:
        return false

    energy.consume(card.cost)
    chain_controller.on_card_played(card.cost)

    card_effect_executor.execute(card)

    deck_controller.move_to_discard(card_id)
    deck_controller.schedule_refill()

    BattleBus.card_played.emit(card_id, card.cost)
    return true
```

## 费用与连锁倍率

规则：

```text
连锁不限制时间。
弃牌中断连锁。
倍率上限暂定 999。
任何费用都可以作为连锁起点，起点倍率为 x1。
只有按费用连续递增才续链，例如 0 → 1 → 2 → 3。
跳费不续链，例如 0 → 2 会从 2 重新开始 x1。
同费不续链，例如 2 → 2 会从第二张 2 重新开始 x1。
后置卡会完全吃到同流派前置卡的效果。
正面数值相关效果按当前链位倍率放大；负面数值效果继承但不放大。
```

推荐实现：

```gdscript
var chain_multiplier: int = 1
var last_cost: int = -1

func on_card_played(cost: int) -> void:
    if last_cost == -1:
        chain_multiplier = 1
        last_cost = cost
        return

    if cost == last_cost + 1:
        chain_multiplier = min(chain_multiplier + 1, 999)
    else:
        chain_multiplier = 1

    last_cost = cost

func break_chain() -> void:
    chain_multiplier = 1
    last_cost = -1
```

例子：

```text
0费 → 1费 → 2费 → 3费
倍率：x1 → x2 → x3 → x4

1费 → 3费 → 4费
倍率：x1 → x1 → x2

2费 → 2费 → 3费
倍率：x1 → x1 → x2
```

## 同名卡限制

规则：

```text
当前版本不限制同名卡连续打出。
同名卡不会因为连续释放进入冷却。
配置了特殊冷却的卡牌仍按自身配置进入冷却。
```

例子：

```text
A → A → A：允许连续打出，不触发冷却
A → B → A → B → A：允许正常打出
```

`same_name_key` 规则：

```text
当前不用于冷却判定。
可保留为同名归类、流派统计或未来共享标记字段。
```

---

## 0 费卡规则

```text
1. 0费卡也可以触发核心技能基准效果。
2. 0费卡可以作为连锁起点。
3. 0费卡后接 1 费卡可以形成 x2。
4. 0费卡不受同名卡连续释放限制。
5. 0费卡也会把自身同流派效果留在当前连锁内，供后置同流派卡继承。
```
