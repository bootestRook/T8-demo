# 开发里程碑与验收清单

## M0：工程骨架

目标：可以进入战斗场景，显示竖屏 UI。

验收：

```text
[ ] Godot 工程目录完成
[ ] BattleRoot.tscn 搭建完成
[ ] BattleHUD 显示城墙血量、能量、等级、经验
[ ] ConfigDB 能加载 CSV
[ ] ObjectPool 可创建和回收 Projectile / Monster
```

---

## M1：基础战斗闭环

目标：怪物按时间点刷新，自动枪械能射击，城墙会受伤。

验收：

```text
[ ] waves.csv 按时间刷怪
[ ] 怪物从上向下移动
[ ] 怪物到达攻击范围后攻击城墙
[ ] 自动枪械锁定近墙怪物
[ ] 枪械有弹匣和换弹
[ ] 怪物死亡给经验
[ ] 城墙死亡战斗失败
```

---

## M2：卡牌基础系统

目标：能抽牌、出牌、消耗能量、弃牌、连锁。

验收：

```text
[ ] 开局从初始牌库随机抽 3 张
[ ] 自动补牌补到 4 张
[ ] 主动抽牌可以让手牌超过 4 张
[ ] 三选一获得新卡：手牌未满 4 张时进手牌，已满 4 张时进牌堆
[ ] 出牌消耗能量
[ ] 出牌后进入弃牌堆
[ ] 空手牌槽自动补牌
[ ] 牌库空时弃牌堆洗回
[ ] 弃牌按钮可用，冷却 20 秒
[ ] 弃牌会中断连锁
[ ] 连锁倍率按费用递增规则增长
[ ] 0费卡不增加倍率
[ ] 同名卡可连续打出且不进入冷却
```

---

## M3：三个核心流派

目标：温压弹、干冰弹、电磁穿刺三套核心技能可玩。

验收：

```text
[ ] 温压弹基准：冲击 + 爆炸 + 击退
[ ] 温压弹词条：连发、爆炸增伤、冲击、范围、引燃、最大生命伤害、总伤害、火花
[ ] 温压弹有 0 费牌

[ ] 干冰弹基准：穿透 3 + 微弱击退
[ ] 干冰弹词条：分裂、贯穿、重冰、增伤、冻结、连发、冻伤、齐射、重冰穿透
[ ] 干冰弹有 0 费牌

[ ] 电磁穿刺基准：单体穿刺 + 麻痹
[ ] 电磁穿刺词条：爆炸、爆炸增伤、爆炸扩张、矩阵、分流、麻痹增伤、裂变、分流+
[ ] 电磁穿刺有 0 费牌
```

## M4：升级三选一

目标：击杀升级后弹三选一，选项能正确应用。

验收：

```text
[ ] 等级上限 20
[ ] 5/10/15/20 级能量上限 +2
[ ] 三选一不出现能量上限
[ ] 三选一可获得新卡牌
[ ] 三选一可强化已有卡牌
[ ] 三选一可出现枪械强化
[ ] 三选一可出现生存强化
[ ] 三选一可出现能量系统强化
[ ] 三选一可出现核心技能基准强化
```

目标：3 分钟精英、5 分钟 Boss 按预期出现。

```text
[ ] 180 秒精英出现

[ ] 8 分钟内可以完成一局
[ ] 后期不会因为弹体过多明显掉帧
[ ] 0费卡不会成为主要伤害来源
[ ] 精简卡组不能无限只打一张牌
```
