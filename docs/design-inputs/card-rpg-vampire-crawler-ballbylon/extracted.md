# 首轮设定提炼稿

## 概念设定

- 概念ID草案：`card-rpg-vampire-crawler-ballbylon`
- 玩法定位：末日僵尸题材卡牌 RPG 技术验证，参考“吸血鬼爬行者”的生存/推进压力、“球比伦战记”的卡牌/构筑体验，以及“皇室战争”的能量限制出牌节奏。
- 平台：手机 Web。
- 目标层级：技术验证，不作为完整游戏首版交付。
- 参考状态：用户表示有参考，但尚未提供截图、链接或更细规则。

## 玩法蓝图

- `hybrid`：卡牌 RPG 与生存推进融合。
- `roguelite`：一局内构筑、随机奖励、战后选择。
- `inventory_backpack`：如果后续参考“球比伦战记”包含格子/装备/组合压力，可作为第二阶段启用。
- `loot_arpg`：如果后续要做装备词条、掉落和成长，可作为第二阶段启用。

## 技术验证主循环

1. 玩家进入一场短战斗，角色和僵尸自动互相攻击或推进。
2. 能量随时间回复，玩家点按手牌消耗能量出牌。
3. 卡牌用于召唤幸存者、设置障碍、造成爆发伤害、治疗或获得护盾。
4. 僵尸持续施压，玩家需要在能量不足和威胁逼近之间做取舍。
5. 击败一波僵尸后出现 3 选 1 奖励，加入牌组或强化属性。
6. 进入下一波，验证奖励会影响后续自动战斗和出牌节奏。
7. 基地或主角生命归零失败，击败指定波次成功，显示结算和重开。

## 本轮只实现

- 手机 Web 可点按操作。
- 1 名玩家、1 种基础敌人、2-3 波战斗。
- 4-6 张基础卡牌：召唤幸存者、路障、燃烧瓶、医疗包、快速射击或能量补给。
- 皇室战争式能量条：能量随时间增长，卡牌有不同能量消耗。
- 战后奖励三选一。
- 简单牌组/弃牌/抽牌状态。
- HUD 显示生命、护盾、敌人血量、当前波次、手牌、结算。

## 暂不实现

- 完整剧情、角色养成、复杂装备词条、长线局外成长。
- 大地图探索、复杂怪物 AI、联网、账号、排行榜。
- 完整正式美术资产库。
