# 玩法总览

## 核心循环

- 时间轴刷怪从屏幕上方向城墙推进。
- 自动枪械锁定距离城墙最近的怪物，提供保底输出、弹匣和换弹节奏。
- 玩家等待能量回复，点击 4 手牌即时释放温压弹、干冰弹或电磁穿刺。
- 正费用卡按费用递增提高连锁倍率，弃牌会中断连锁。
- 击杀获得经验，升级时暂停并弹出三选一。
- 3:00 精英怪检验控制和爆发，5:00 Boss 死亡胜利，城墙归零失败。

## 首版范围

- 目标层级：完整首版可试玩 demo。
- 内容单元：基础防线、精英压迫、Boss 检定。
- 首版只做三流派：温压弹、干冰弹、电磁穿刺。
- 枪械不进入卡牌系统，只作为自动输出和升级强化存在。

## 启用系统

- 时间轴刷怪
- 城墙生命和失败
- 自动枪械、弹匣、换弹、近墙优先目标
- 能量回复和等级固定能量上限
- 4 手牌、弃牌、补牌、牌库/弃牌堆循环
- 费用连锁倍率和同名卡连续冷却
- 温压弹、干冰弹、电磁穿刺三流派
- 升级三选一
- 胜负结算、重开、本地进度

## 系统边界

- 不做背包格子、装备掉落、联网账号、剧情养成和复杂状态机。
- 第一版配置集中在 `ContentUnits.gd`，后续可迁移到 CSV/ConfigDB。
- 程序化绘制是占位美术，后续正式素材必须进入 `assets/` 并登记。
