# UI 与 HUD

## HUD 信息

- 顶栏：参考竖屏塔防卡牌界面，使用玻璃态分块显示波次、防线守卫进度、金币和晶体。
- 战斗区：尽量少遮挡核心单位；可显示程序化推荐落点、压线警告和路标。
- 底部状态：基地 HP、压线警告、牌库、刷新、手牌数量、可合成提示和能量条。
- 底部操作：4 张扇形手牌按钮、弃牌按钮、升级三选一按钮、重新开始按钮。
- 中央提示：当前事件、升级三选一提示、胜利/失败结算。

## 手机触屏规则

- 手牌按钮始终在底部安全区，单张按钮显示序号、卡名、费用和简短效果。
- 能量不足时手牌按钮置灰。
- 升级三选一时隐藏手牌，避免误触。
- 胜负后只保留重新开始入口。
- 顶栏和底栏必须避开主战斗点击区；普通战斗中不显示大中心面板。
- 手牌扇形叠放，但费用、卡名和点击状态必须可读。

## UI 素材

- 当前状态：程序化占位 UI。按钮、面板、战术提示圈和警告标识由 `Hud.gd` / `Game.gd` 生成 StyleBox 或绘制图形，不计入正式美术素材。
- 后续正式素材：卡牌底板、弃牌按钮、升级选项面板、城墙生命条、三流派图标，进入 `assets/ui/` 并登记。
