# UI 与 HUD

## HUD 信息

- 顶栏：剩余时间、城墙生命、能量、连锁、等级、击杀、分数。
- 中央提示：当前事件、升级三选一提示、胜利/失败结算。
- 底部操作：4 张手牌按钮、弃牌按钮、升级三选一按钮、重新开始按钮。

## 手机触屏规则

- 手牌按钮始终在底部安全区，单张按钮显示序号、卡名、费用和简短效果。
- 能量不足时手牌按钮置灰。
- 升级三选一时隐藏手牌，避免误触。
- 胜负后只保留重新开始入口。

## UI 素材

- 当前状态：程序化占位 UI。按钮和面板由 `Hud.gd` 生成 StyleBox，不计入正式美术素材。
- 后续正式素材：卡牌底板、弃牌按钮、升级选项面板、城墙生命条、三流派图标，进入 `assets/ui/` 并登记。
