# 内容单元

当前 Demo 的内容单元来自 `assets/data/combat/level_configs.json` 与 `assets/data/combat/waves.json`。每个关卡 20 波，基础目标都是守住防线并清场，差异来自怪物组合、波次密度、关卡系数、节奏和 Boss 压力。

## 关卡 1：1-1 教堂广场

- 目标：守住教堂广场，清完 20 波怪物。
- 差异：入门节奏，血量系数 0.5，主要使用基础小怪和快速怪。
- 压力：低数值压力，重点让玩家理解手牌、连锁、弃牌和防线生命。
- 素材：当前使用正式战场、玩家、基础怪/快速怪、HUD 与卡牌 UI 资源。

## 关卡 2：1-2 教堂侧廊

- 目标：守住教堂侧廊，清完 20 波怪物。
- 差异：血量系数提升到 0.88，波次组合比第 1 关更紧。
- 压力：更高生存压力，玩家需要更稳定地续连锁和使用防线类卡。
- 素材：复用当前运行时战场、怪物和 HUD 资源。

## 关卡 3：1-3 教堂中庭

- 目标：守住教堂中庭，清完 20 波怪物。
- 差异：攻击系数 1.1，血量系数 1.25，怪物耐久和推进威胁继续提升。
- 压力：更强调目标优先级、控制卡和伤害链的节奏。
- 素材：复用当前运行时资源。

## 关卡 4：1-4 钟楼前线

- 目标：守住钟楼前线，清完 20 波怪物。
- 差异：攻击系数 1.3，血量系数 1.63，精英压力更明显。
- 压力：防线损耗更快，玩家需要在出牌、弃牌和升级奖励之间做取舍。
- 素材：复用当前运行时资源。

## 关卡 5：1-5 教堂广场决战

- 目标：守住教堂广场决战，清完 20 波怪物。
- 差异：攻击系数 1.5，血量系数 2.13，第 16 波包含 Boss。
- 压力：Boss 波是当前 Demo 的高峰检查点，要求玩家在前中期积累足够连锁和升级收益。
- 素材：使用当前 Boss、精英怪、普通怪、战场和 HUD 资源。
